🛠️ Procesador de Marquillas para Proyectos de Instrumentación y Control

Optimización y automatización de etiquetas para cableado industrial (As-Built).

Este proyecto nace de la necesidad real en campo de reducir los tiempos de oficina técnica y planeación. Es una herramienta web ligera diseñada para ingenieros, instrumentistas y planners de obra que gestionan grandes volúmenes de datos de cableado y necesitan formatos de impresión listos para etiquetadoras industriales.

🚀 ¿Qué problema resuelve?

El formateo manual de listados de cables en Excel para generar marquillas es una tarea propensa a errores y consume horas de trabajo. Esta aplicación automatiza el proceso de:

Descombinar celdas: Limpia archivos exportados de bases de datos de ingeniería.

Relleno automático (Fill-Down): Hereda los valores anteriores para completar filas vacías.

Estructuración lógica: Concatena TAGS, Orígenes y Destinos siguiendo la lógica técnica necesaria para el montaje.

✨ Características Principales

Modo Automático: Configurado para la estructura estándar TAGNAME : DESDE : BORNERA 1 : BORNE 1.

Constructor de Bloques Personalizado: Interfaz gráfica de arrastrar y soltar (Drag & Drop) para armar marquillas a medida.

Privacidad Total: Los datos nunca salen de tu computadora. El procesamiento se realiza 100% en el navegador (local).

Salida Profesional: Exporta a .xlsx con la propiedad "Ajustar texto" activada automáticamente, lista para impresión.

🛠️ Tecnologías Utilizadas

HTML5 & JavaScript (Vanilla): Sin dependencias pesadas para mayor velocidad.

Tailwind CSS: Interfaz moderna y adaptativa.

SheetJS & xlsx-js-style: Para la manipulación avanzada de libros de Excel y aplicación de estilos nativos.

📖 Instrucciones de Uso

Sube tu archivo de Excel o CSV (ej. Listado de Cables).

Selecciona el modo de trabajo (Automático o Personalizado).

Asocia las columnas de tu archivo a los campos requeridos.

¡Descarga tu archivo listo!
